# Глава 14: Сравнение отчётов (Diff)

Сравнение двух отчётов отвечает на вопрос: «Стало лучше или хуже?»

## Что сравнивается

- **USE-метрики**: утилизация, насыщение, ошибки (delta > 5% = значимое)
- **Гистограммы**: P99 латентность
- **Аномалии**: новые vs исчезнувшие

## Классификация

- **Stable**: delta < 5%
- **Improvement**: delta < -5%
- **Regression**: delta > 5%

```bash
sysdiag diff baseline.json current.json --json
```
