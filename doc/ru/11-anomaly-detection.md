## Шкала здоровья

Оценка здоровья (0-100) рассчитывается из обнаруженных аномалий с весами по категориям:
- **CPU**: вес 1.5x
- **Память**: вес 1.5x
- **Диск**: вес 1.0x
- **Сеть**: вес 1.0x
- **Контейнер**: вес 1.0x

Критическая аномалия = -20 баллов, Warning = -10 баллов (с учётом веса).
